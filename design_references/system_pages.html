Налоговые ставки

v1.2.0
GET /api/v1/tax/tax-rates

Получение информации о налоговых ставках для различных категорий доходов и налогоплательщиков.

Параметры запроса

Параметр Тип Обязательный Описание
country string Опционально Код страны в формате ISO 3166-1 alpha-2. По умолчанию "RU".
year integer Опционально Год, для которого запрашиваются налоговые ставки. По умолчанию текущий год.
category string Опционально Категория налога: "income", "property", "business", "all". По умолчанию "all".

Пример запроса

cURL bash 
# Запрос налоговых ставок по доходам для России на 2023 год
curl -X GET "https://api.dohodometr.ru/api/v1/tax/tax-rates?country=RU&year=2023&category=income" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

GET /api/v1/tax/tax-rates/{country}/history

Получение исторических данных об изменении налоговых ставок для указанной страны.

Параметры пути

Параметр Тип Обязательный Описание
country string Обязательно Код страны в формате ISO 3166-1 alpha-2.

Параметры запроса

Параметр Тип Обязательный Описание
from_year integer Опционально Начальный год для исторических данных. По умолчанию 10 лет назад.
to_year integer Опционально Конечный год для исторических данных. По умолчанию текущий год.
category string Опционально Категория налога: "income", "property", "business", "all". По умолчанию "income".

Содержание

  • GET /tax-rates
  • GET /tax-rates/{country}/history
  • GET /tax-rates/calculate
  • GET /tax-rates/deductions
